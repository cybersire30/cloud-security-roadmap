About This Repository
This repository documents my self-directed learning journey into cloud security, starting from a foundation in ethical hacking (6-month internship) and building toward a junior Cloud Security Engineer role by graduation in December 2026.
Study schedule: 30–45 mins on weekdays | 1 hour on weekends | 7 days/week (~6.5 hrs/week)
Why cloud security? Cloud security sits at the intersection of security thinking and modern infrastructure — it rewards an attacker mindset (which I already have) without being as code-heavy as traditional ethical hacking roles.
🛠️ Skills Being Built
Cloud Platforms: AWS (primary), Azure & GCP (awareness)
Security Tools: GuardDuty, Security Hub, CloudTrail, CloudWatch, CloudGoat
Concepts: IAM, VPC Security, Zero Trust, CSPM, Threat Detection, MITRE ATT&CK Cloud
Background: Ethical Hacking, Network Security, Penetration Testing

📌 How to Navigate This Repo
Each phase folder has its own README.md with detailed progress tracking. Project folders contain architecture diagrams, setup instructions, security decisions made, and lessons learned. Notes folders contain weekly study summaries written in plain language — the act of writing these is part of the learning process.
